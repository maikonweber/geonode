{% trans "About GeoNode" %}

O GeoNode é um sistema de gerenciamento de conteúdo geoespacial, uma plataforma para o gerenciamento e publicação de dados geoespaciais. Ele reúne projetos de software de código aberto maduros e estáveis ​​sob uma interface consistente e fácil de usar, permitindo que usuários não especializados compartilhem dados e criem mapas interativos.

As ferramentas de gerenciamento de dados integradas ao GeoNode permitem a criação integrada de dados, metadados e visualizações de mapas. Cada conjunto de dados no sistema pode ser compartilhado publicamente ou restrito, permitindo o acesso apenas a usuários específicos. Recursos sociais, como perfis de usuário e sistemas de avaliação, possibilitam o desenvolvimento de comunidades em torno de cada plataforma, facilitando o uso, o gerenciamento e o controle de qualidade dos dados contidos na instância do GeoNode.

Ela também foi projetada para ser uma plataforma flexível que os desenvolvedores de software podem estender, modificar ou integrar para atender aos requisitos de seus próprios aplicativos.
